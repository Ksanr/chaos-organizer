## О проекте

Chaos Organizer — веб-приложение в стиле мессенджера (Telegram / WhatsApp), где бот-органайзер помогает хранить информацию, искать её, закреплять важные сообщения, отправлять файлы, геолокацию и многое другое.

**Демо-данные:** при первом запуске сервера в чате уже есть несколько примеров сообщений, чтобы вы могли сразу оценить интерфейс.

---

## Реализованные функции

### Обязательные

| Функция                      | Описание                                                                                                                                                                                                              |
|------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Текстовые сообщения и ссылки** | Отправка обычного текста. Любая ссылка (http:// или https://) становится кликабельной и открывается в новой вкладке.                                                                                                  |
| **Загрузка файлов**  (изображения, видео, аудио) | Изображения, видео и аудио загружаются через кнопку «📎» или перетаскиванием (Drag & Drop) в область чата. <br/> * Данная функция работает только локально, сервер Pxxl не выдаёт ошибки при активации данной функции |
| **Скачивание файлов**        | Для любого загруженного файла доступно скачивание на компьютер (по клику на имя файла).<br/> * См. выше                                                                                                               |
| **Ленивая подгрузка**        | Сначала показываются последние 10 сообщений. При прокрутке вверх подгружаются следующие 10 и так далее.                                                                                                               |

### Дополнительные (5 шт.)

| # | Функция | Как пользоваться |
|---|---------|------------------|
| 1 | **Синхронизация между вкладками** | Откройте приложение в двух вкладках. Сообщение, отправленное в одной, сразу появится во второй (через `BroadcastChannel API`). |
| 2 | **Поиск по сообщениям** | Введите слово в поле поиска (внизу чата) — выпадет список сообщений (текстовых и файлов), содержащих этот текст. Клик по результату прокрутит чат к сообщению. |
| 3 | **Отправка геолокации** | Нажмите кнопку «📍». Разрешите доступ к местоположению — в чат отправится сообщение с координатами. |
| 4 | **Закрепление сообщений (pin)** | У каждого сообщения есть значок «📌». Клик закрепляет сообщение — оно появляется вверху чата. Повторный клик снимает закрепление. Закреплено может быть только одно сообщение. Открепить можно также по кнопке внутри виджета. |
| 5 | **Поддержка emoji** | Нажмите кнопку «😀» — откроется панель выбора смайлов. Клик по смайлу вставляет его в поле ввода. Панель автоматически закрывается после отправки сообщения. |

## Технологии

- **Фронтенд:** JavaScript (ES6+), Webpack, Babel, ESLint, CSS3.
- **Бэкенд:** Node.js, Koa, `@koa/router`, `@koa/cors`, `koa-bodyparser`, `@koa/multer`, `uuid`.
- **Деплой:**  
  - Фронтенд — GitHub Pages (ветка `gh-pages`).  
  - Бэкенд — Pxxl.app (бесплатный хостинг, но есть лимит по объёму, что не позволяет показать работу с файлами).

## Запуск локально

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/Ksanr/chaos-organizer.git
cd chaos-organizer
```
2. **Запустите бэкенд** (порт 3000)
```bash
cd backend
npm install
npm run dev
```
3. **В другом терминале запустите фронтенд** (порт 8080)
```bash
cd frontend
npm install
npm start
```
4. **Откройте http://localhost:8080**
```bash
Убедитесь, что порты 3000 и 8080 не заняты.
```
